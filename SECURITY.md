# Security Policy

## Supported Versions

Only the latest `stable` version of Spatial OS receives security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | ✅ Supported        |
| < 1.0   | ❌ Not Supported    |

## Reporting a Vulnerability

We take the security of Spatial OS seriously. If you believe you have found a security vulnerability, please do **not** create a public issue. Instead, follow these steps:

1. Send an email to **security@spatial-os.org**.
2. Include details of the vulnerability.
3. Provide a Proof of Concept (PoC) if possible.
4. We will acknowledge your report within 48 hours and provide a timeline for a fix.

We follow a coordinated disclosure policy. Please allow us time to fix the issue before sharing details publicly.
